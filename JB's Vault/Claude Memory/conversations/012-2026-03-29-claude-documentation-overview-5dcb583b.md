Claude Code is an agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster by executing routine tasks, explaining complex code, and handling git workflows — all through natural language commands.

You can use it across multiple surfaces: the full-featured terminal CLI, VS Code, a web interface, iOS/Android, and the Desktop app. Sessions aren't tied to a single surface — you can start a task in the terminal and continue it from your phone, or hand off a terminal session to the Desktop app for visual diff review.

It also supports scheduled tasks — cloud-based ones run on Anthropic infrastructure even when your computer is off, while desktop scheduled tasks run on your machine with direct access to local files and tools.

Recent additions include a Windows PowerShell preview tool, richer admin and MCP controls, faster startup and caching, and channel-based permission relays that can forward tool approval prompts to your phone.
