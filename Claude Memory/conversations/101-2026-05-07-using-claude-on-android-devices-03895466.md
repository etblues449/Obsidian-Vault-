# Using Claude on Android devices

> 6 messages · 2026-05-07

### Human — 2026-05-07T01:19

How do I get claude.ai on my android phone

### Assistant — 2026-05-07T01:19

You can get Claude on Android by downloading the official Claude app from the Google Play Store. Search for "Claude" by Anthropic, or go directly to: https://play.google.com/store/apps/details?id=com.anthropic.claude

Once installed, sign in with the same account you use on claude.ai and your conversations will sync across devices.

### Human — 2026-05-07T01:20

Not that app. Claude desktop

### Assistant — 2026-05-07T01:20

Claude Desktop isn't available for Android — it's only released for Windows and macOS.

For Android, your options are:
1. The Claude mobile app (from Play Store)
2. claude.ai in your mobile browser

If you specifically want desktop-style features on Android, the browser version of claude.ai is the closest match.

### Human — 2026-05-07T01:21

When I use claude.ai in browser goes back into the app

### Assistant — 2026-05-07T01:21

That's the Android app intercepting claude.ai links. To force the browser to open it instead:

**Option 1: Disable link handling in the Claude app**
1. Long-press the Claude app icon → App info (or Settings → Apps → Claude)
2. Tap "Open by default"
3. Turn off "Open supported links" / clear defaults

**Option 2: Use an incognito/private tab**
Open Chrome → new Incognito tab → type claude.ai. Incognito tabs usually ignore app-link associations.

**Option 3: Request desktop site**
In Chrome, tap the three dots → check "Desktop site" before navigating.

Try Option 1 first — that's the permanent fix.
